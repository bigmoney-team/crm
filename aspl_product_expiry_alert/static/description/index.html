Odoo 12 Product Serial/Lot Expire Dashboard

This module allows to product alert to going expire.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • This module allow to view all the product that are going to expire by location and warehouse.
  • you can search the location and warehouse that display all the products for that location or warehouse.
  • Also allow the show near expiry product graph to select custome date.
  • Allow to add near expire product blocks with configuration.
  • Send email of near expire product detail based on configuration.

How to use

  • In Sales configuration setting, enable option variants and options.

  • In Inventory configuration setting, enable option 'Lots & Serial Numbers', 'Storage Locations' and 'Multi-Warehouses'.

  • In Tracking ==> Select by lots/ by unique serial number.

  • Click on 'Update qty on hand' button to update a quantity of particular product.

  • By clicking on "Start Inventory" button start inventory adjustment.

  • You can select a location, update quantity and select or click on create and edit to create a new lot.

  • You can create a lot and select end of life date and alert date.

  • In Inventory configuration setting ==> Go to 'Product Expiry Configuration' menu.

  • Create a custom near expire product block.

  • Product Expiry Dashboard.

  • Based on select date range bar graph and pie chart view will be generated.


  • You can search a category, warehouse and loaction.

  • Product expire mail configuration.

  • Mail template for near Expire Product.
